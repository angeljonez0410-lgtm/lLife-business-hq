# Life + Business HQ

A bold, girly, luxury “boss CEO” dashboard web app to organize your life, schedule, learning, tasks, content, products, money, and all your businesses.

## Tech Stack
- Next.js (App Router, TypeScript)
- Tailwind CSS
- shadcn/ui
- Lucide icons
- Framer Motion
- Supabase (backend/auth/database)
- Mobile-friendly, Vercel-ready

## Features
- Dashboard Home: Command center, daily focus, motivational widgets
- Business Hub: Pages for each business (overview, goals, tasks, learning, content, products, notes, income/expenses)
- Task Manager: Prioritized, categorized, and filtered tasks
- Weekly Schedule Planner: Drag-and-drop weekly focus
- Content Planner: Multi-platform content hub with AI prompts
- Money Tracker: Income/expense tracking and profit summary
- Product Tracker: For Amazon, dropshipping, crafts, and shirts
- Learning Mode: Beginner-friendly learning center
- Low/High Hustle Modes: Focused or batch work views
- AI Assistant: Chat UI for "Zu" (business, content, motivation)
- Admin/Settings: Profile, categories, theme, onboarding

## Getting Started
1. `npm install`
2. `npm run dev`
3. Visit [http://localhost:3000](http://localhost:3000)

## Deployment
- Ready for Vercel
